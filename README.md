# Rotating/swapping tiles puzzle

A web-based rotating/swapping tiles puzzle generator. Each tile has colored shapes on its sides that must match its neighbors. Tiles can be dragged, swapped, and rotated. There is always at least one solution.

## Controls
- **Width/Height**: Set the puzzle grid size
- **Regenerate**: Create a new puzzle with the current size
- **Solve**: Solve one tile (position or rotation) per click
- **Drag/Drop**: Swap tiles
- **Click/Double-tap**: Rotate a tile 90Â° clockwise
